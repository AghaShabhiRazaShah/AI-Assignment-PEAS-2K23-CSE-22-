# AI-Assignment-PEAS-2K23-CSE-22-
# Fraud Detection System for Digital Payments and Banking Transactions

## Project Overview
This project is an AI-based Fraud Detection System that identifies suspicious banking and digital payment transactions.

## Objective
To detect fraudulent transactions and protect users from financial losses.


# PEAS Specification

## Performance Measure
- Fraud detection accuracy
- False positive rate
- Transaction processing speed
- Customer satisfaction

## Environment
- Online banking systems
- Mobile payment applications
- ATM transactions
- Credit/Debit card networks

## Actuators
- Fraud alerts
- Transaction blocking
- Customer notifications
- Security reports

## Sensors
- Transaction history
- User login data
- Device information
- Location tracking
